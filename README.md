# 通讯录前端（原生 HTML + JS）

## 说明
- 纯静态页面，通过 `fetch` 调用后端 REST API
- 可使用 VSCode Live Server 或任意静态服务打开

## 运行
1) 启动后端 `http://localhost:8080`
2) 以静态服务器打开本目录（例如 VSCode Live Server）
3) 在页面顶部配置后端 API 地址（见 `src/config.js`）

## 功能
- 列表联系人
- 新增联系人
- 修改联系人
- 删除联系人
- 分类联系人

## 结构
```
通讯录.html
src/
  config.js
  app.js
```
